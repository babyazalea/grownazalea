* Vue3 + Typescript 기반의 단일 페이지 어플리케이션(SPA, single page application)입니다.
* 국립중앙도서관의 서지 정보 API를 활용했습니다. 각 출판사에서 직접 등록한 출간 예정 도서를 검색할 수 있습니다.
* [Pinia]("https://pinia.vuejs.org/")(Vuex와 거의 똑같은 기능)를 이용하여 전역 상태를 관리합니다.
* Vercel로 배포 되었습니다.
